# Tests

Place your test files in this directory or alongside the files they test.

## Where to Put Tests

You have two options:

1. **In this directory**: `__tests__/ComponentName.test.tsx`
2. **Next to the file**: `components/layout/Navbar.test.tsx`

Both approaches work — pick one and be consistent across your team.

## Running Tests

```bash
pnpm test          # Run in watch mode
pnpm test:ui       # Open the Vitest UI
pnpm test:coverage # Generate coverage report
```
