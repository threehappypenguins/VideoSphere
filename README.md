# VideoSphere

**Upload once, distribute everywhere.**

VideoSphere is an open-source, self-hosted video distribution platform. Upload a video once to Cloudflare R2, then distribute it to YouTube, Vimeo, and optional destinations such as Google Drive, SFTP, or SMB backup. Metadata drafting and AI-assisted descriptions help you prepare uploads before they go live.

VideoSphere runs as a Docker container you control — suitable for homelabs, churches, and small teams who want their own infrastructure.

## Documentation

Full documentation lives at **[videosphere.sarahpoulin.ca](https://videosphere.sarahpoulin.ca/)**.

To deploy a pre-built container in production, see the **[Deployment Guide](https://videosphere.sarahpoulin.ca/deployment-guide.html)**.
